# 默认提示词

本项目采用 bun 开发，各种操作的命令参见 package.json。修改代码后，请跑一遍 `test:all`；如果需要，请修改 package.json 和 README.md。

## 前端

请你通过 shadcn/ui 安装全部 ui 组件，而不是自己写（当然，如果 shadcn/ui 不提供，你也可以自己写，保持风格一致即可），并采用相对简洁紧凑的 ui 风格（即 shadcn/ui 官方的风格）。

不要相信用户传入的内容，一定记得进行合法性判断。

在所有汉字和半角字符之间插入空格，全角标点和半角字符之间不需要插入。

不要画蛇添足地添加任何完全不必要的元素。

## 后端

请遵循 RESTful API 规范。

不要相信前端传回的内容，一定记得进行合法性判断。

## 代码风格

请采用简洁且易于维护的代码风格，尽量不写注释，不要重复造轮子。
